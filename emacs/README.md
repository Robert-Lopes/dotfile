# Sobre as configurações do Emacs

Durante a criação do arquivo `config_theme.sh`, que automatiza a customatização do emacs, faltou alguns pontos importantes que precisam ser levados em consideração para a execução eficiente do script. Sendo elas:

- [ ] A verificação e o tratamento correto caso o emacs não esteja instalado;
- [ ] Verificar se a pasta `theme` existe dentro do diretório `emacs.d;`
- [ ] Se a pasta `theme` não existir, deve-se criar a pasta;
- [ ] Verificar se os caminhos estão corretos;
- [ ] Instalar os arquivos do theme dentro da pasta `theme`;
